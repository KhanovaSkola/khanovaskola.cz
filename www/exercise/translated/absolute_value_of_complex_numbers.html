Určete absolutní hodnotu následujících komplexních čísel:

REPRESENTATION

ABS_SQUARE

Absolutní hodnota libovolného čísla je jeho vzdálenost od nuly. Jelikož komplexní čísla mohou být zobrazena jako body na komplexní rovině, absolutní hodnota komplexních čísel může být určena pomocí vzorce pro vzdálenost dvou bodů.

graphInit({ range: [[-10, 10], [-10, 10]], scale: 20, tickStep: 1, labelStep: 1, }); label( [10, 0.5], "Re", "left" ); label( [0.5, 9], "Im", "right" ); circle( [REAL, IMAG], 3 / 20, { fill: KhanUtil.BLUE, stroke: "none" }); label( [REAL, IMAG], REPRESENTATION, "left", { color: KhanUtil.BLUE, labelDistance: 10 } );

REPRESENTATION je v grafu zobrazena jako modrý bod.

path([ [0,0], [REAL, IMAG]], { stroke: KhanUtil.ORANGE });

Absolutní hodna kterou hledáme je délka oranžové úsečky.

path([ [0,0], [REAL, 0], [REAL, IMAG]], { stroke: KhanUtil.BLUE });

Oranžová úsečka je přepona pravoúhlého trojúhelníku. Jeho dvě odvěsny (nakreslené modře) mají délky abs( REAL ) a abs( IMAG ), což odpovídá absolutním hodnotám reálné a imaginární části komplexního čísla REPRESENTATION.

Doplněním do Pythagorovy věty:
\qquad |REPRESENTATION|^2 = abs( REAL )^2 + abs( IMAG )^2, takže
\qquad |REPRESENTATION| = \sqrt{abs( REAL )^2 + abs( IMAG )^2}.

\qquad \sqrt{abs(REAL)^2 + abs(IMAG)^2} = \sqrt{REAL * REAL + IMAG * IMAG} = \sqrt{ABS_SQUARE}

Zjednodušením získáme formattedSquareRootOf( ABS_SQUARE ). To je absolutní hodnota REPRESENTATION.

Dále už výraz nemůžeme zjednodušit. Absolutní hodnota REPRESENTATION je \sqrt{ABS_SQUARE}.
